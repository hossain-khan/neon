# Web App

`webApp` is the browser demo shell for Neon.

## What lives here

- JS and Wasm executable targets
- browser entrypoints for the shared showcase
- webpack/browser packaging for demo deployment

## Depends on

- `shared`
- Compose UI

## Targets

- JS browser
- WasmJs browser

## Purpose

This module exists to validate Neon in browser environments and to host the showcase for web targets.
