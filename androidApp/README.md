# Android App

`androidApp` is the Android demo shell for Neon.

## What lives here

- Android application packaging
- Android-specific bootstrap for the shared showcase
- direct Android wiring needed by `engine-highlightjs`

## Depends on

- `shared`
- `engine-highlightjs`
- Android Activity Compose

## Entry point

- `MainActivity`

## Purpose

This module exists to run and validate the Neon showcase on Android. It is not a reusable library module.
