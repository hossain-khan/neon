# Desktop App

`desktopApp` is the JVM desktop demo shell for Neon.

## What lives here

- Compose Desktop application packaging
- desktop window entrypoint for the shared showcase

## Depends on

- `shared`
- Compose Desktop
- Swing coroutines support

## Entry point

- `dev.hossain.neon.MainKt`

## Purpose

This module exists to run and package the Neon showcase on desktop platforms.
